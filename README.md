# Ayur--phenotype-detector
Ayur is a web-based chatbot that helps users identify their dominant Ayurvedic dosha (Vata, Pitta, or Kapha) through an interactive questionnaire. The application analyzes user responses and determines the most suitable body constitution based on Ayurvedic principles.

# Features
Interactive chatbot interface
Dosha assessment (Vata, Pitta, Kapha)
Responsive design
Dark/Light mode support
Fast and user-friendly experience

# Technologies Used
HTML
CSS
JavaScript
JSON

# How It Works
Users answer a series of questions related to their physical and mental characteristics. Each response is mapped to a dosha, and the dosha with the highest score is displayed as the final result.
